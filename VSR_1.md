# ВСР 1

## Обзор актуальных инструментов для оперативного прототипирования интерфейсов IT-продуктов. Подготовка презентации по наиболее эффективному решению и его публичная защита.

### Adobe XD

Adobe XD является одним из наиболее востребованных инструментов в сфере UI/UX-дизайна. Платформа предназначена для разработки вайрфреймов, интерфейсов веб-сайтов и мобильных приложений, а также голосовых интерфейсов. Инструмент поддерживает совместную работу в режиме реального времени, что позволяет командам оперативно получать обратную связь от заказчиков и участников проекта.

### Figma

Figma представляет собой удобный онлайн-редактор с поддержкой drag-and-drop, который широко используется для создания вайрфреймов и интерактивных прототипов. Инструмент знаком большинству дизайнеров и сочетает в себе возможности проектирования UX и быстрого прототипирования. Дополнительный функционал реализуется с помощью плагинов, таких как Figmotion и Autoflow, а также инструментов работы с векторной графикой.

### Webflow

Webflow активно набирает популярность благодаря сочетанию возможностей дизайна и разработки. Платформа позволяет создавать интерактивные прототипы, которые одновременно являются полноценными веб-сайтами. Webflow ориентирован на разработку сайтов без написания кода, предоставляя при этом семантически корректную разметку и инструменты для создания сложных дизайнерских решений с последующей публикацией.

### Sketch

Sketch — один из самых распространённых инструментов для проектирования пользовательских интерфейсов. Он используется для создания различных типов UI и прототипов и во многом схож с Figma по функциональности. Основным ограничением Sketch является доступность только для операционной системы macOS.

### InVision

InVision — веб-платформа, ориентированная на процесс проектирования интерфейсов и прототипирования. Инструмент охватывает весь путь от создания вайрфреймов до финального UI-дизайна и поддерживает эффективное командное взаимодействие. Платформа позволяет вносить изменения в режиме реального времени и оперативно собирать отзывы, что упрощает организацию проектной деятельности.

### Axure RP

Axure RP объединяет мощные средства проектирования с возможностями создания высокоуровневых интерактивных прототипов. Платформа поддерживает импорт SVG, интеграцию со Sketch и Adobe XD, а также совместную работу через Axure Cloud. Дополнительно Axure RP предоставляет инструменты для автоматизированной документации и передачи материалов разработчикам без необходимости программирования.

### Framer

Framer отличается гибкостью и расширенными возможностями настройки интерактивных прототипов. В отличие от классических инструментов с визуальным редактором, платформа основана на использовании кода и применяет CoffeeScript для создания сложных сценариев взаимодействия. К недостаткам Framer можно отнести высокий порог входа и необходимость дополнительного времени на освоение.

### Origami Studio

Origami Studio — инструмент прототипирования для macOS, разработанный компанией Facebook. Он активно используется внутри компании и позволяет создавать интерактивные прототипы с возможностью просмотра на мобильных устройствах. В сравнении с предыдущими версиями производительность инструмента была значительно улучшена, однако его использование по-прежнему ограничено экосистемой Apple.

### Marvel

Marvel позиционируется как платформа для проектирования цифровых продуктов и отлично подходит для создания прототипов мобильных и веб-приложений. Инструмент предоставляет набор шаблонов и средств для визуализации пользовательских сценариев, а также поддерживает анимацию и работу с жестами, что делает его удобным для быстрого прототипирования.

### Justinmind

Justinmind — инструмент, ориентированный на разработку прототипов с акцентом на пользовательский опыт. Платформа поддерживает создание адаптивных вайрфреймов и интерактивных макетов для различных разрешений экранов. Широкий набор шаблонов и UI-компонентов позволяет создавать детализированные прототипы без необходимости написания программного кода.
