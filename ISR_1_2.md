# ИСР 1.2

Разработка диаграммы Ганта для планирования и координации проектной деятельности, связанной с созданием компонента образовательной среды для учебного заведения. Дополнительно выполняется анализ возможных рисков проекта, а также формирование отчётного репозитория на платформе GitHub с корректным оформлением документации с использованием разметки Markdown.

![Диаграмма Ганта проекта образовательной среды](https://user-images.githubusercontent.com/31900142/211079752-4e1e4936-afb4-409f-9325-e9de65446d20.png)

## Анализ рисков проекта

- Возможное отрицательное восприятие идеи со стороны педагогического коллектива  
- Вероятность несогласия или критики проекта со стороны родителей  
- Недостаточный уровень профессиональной подготовки педагогов для работы с новым компонентом  
- Потеря актуальности проекта в процессе его реализации  
- Нарушение запланированных сроков разработки  
- Риск отсутствия финансирования со стороны администрации муниципального района
